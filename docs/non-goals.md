# Non-Goals (MVP Exclusions)

The following items are explicitly out of scope for the MVP:

- **Bluetooth device control** (smart trainer control, HR monitor pairing).
- **Full auto-planner** (no fully automated multi-week plan generation).
- **Marketplace features** (coach marketplace, shared plans store).
- **Advanced social features** (leaderboards, team training).
- **Real-time workout execution** (live workout player).
- **Multi-user/enterprise features** (teams, org billing, admin dashboards).
